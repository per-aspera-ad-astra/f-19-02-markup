Tabs (fade/slide effect)

This jQuery plugin provides tabs switching with/without effect. Flexible width and autorotation are also supported.

The following options are available in the plugin:

		jQuery('ul.tabset').contentTabs({
			activeClass:'active',      // active class for active tab navigation link
			addToParent:false,         // add the class not to the link but to its parent
			autoHeight:false,          // smooth changing of the height during tabs switching (first parent node is used)
			autoRotate:false,          // autorotation on/off
			switchTime:3000,           // autorotation frequency (3s)
			animSpeed:400,             // animation speed
			effect: 'none',            // effect for switching the tabs "none", "fade" or "slide"
			tabLinks:'a',              // selector of tab navigation links in the tabset
			event:'click'              // switch on event "click", "mouseover", etc.
		});
